# Employee-Traker

## Description  
    A command-line application to manage a company's employee database, using Node.js, Inquirer, and MySQL.
   
  ## Installation 

      npm init
      npm inquirer
      MYSQL
      

  ## Usage 

      by running node server on the terminal screen
